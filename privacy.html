개인정보 처리방침

📅 시행일: 2025년 12월 1일

고순이(이하 "회사")는 「개인정보 보호법」에 따라 이용자의 개인정보를 보호하고 이와 관련한 고충을 신속하게 처리하기 위하여 다음과 같이 개인정보 처리방침을 수립·공개합니다.

1. 수집하는 개인정보 항목

📱 로컬 저장 방식

본 게임은 서버를 운영하지 않으며, 게임 데이터는 사용자 기기에 로컬 저장됩니다.

단, 아래 제3자 서비스를 통해 다음 정보가 수집될 수 있습니다:

광고 서비스 (Unity Ads)

  • 기기 정보 (모델명, OS 버전)
  • 광고 식별자 (GAID/IDFA)
  • IP 주소 (대략적 위치 추정용)
  • 광고 시청 기록

광고 서비스 (Google AdMob)

  • 기기 정보 (모델명, OS 버전)
  • 광고 식별자 (GAID/IDFA)
  • IP 주소 (대략적 위치 추정용)
  • 광고 상호작용 데이터

분석 서비스 (Firebase Analytics)

  • 기기 정보 (모델명, OS 버전)
  • 앱 사용 통계 (세션, 이벤트)
  • 대략적 위치 (IP 기반)

2. 개인정보 수집 목적

서비스 목적
Unity Ads 무료 게임 제공을 위한 광고 수익
Google AdMob 무료 게임 제공을 위한 광고 수익
Firebase Analytics 서비스 개선 및 통계 분석

3. 개인정보 보유 및 이용 기간

회사는 직접 개인정보를 저장하지 않습니다.

제3자 서비스의 보유 기간은 각 서비스의 정책을 따릅니다:

  • Unity 개인정보 처리방침
  • Google/Firebase/AdMob 개인정보 처리방침

4. 개인정보의 제3자 제공

제공받는 자 제공 항목 제공 목적
Unity Technologies 기기정보, 광고ID, IP 광고 제공
Google (AdMob) 기기정보, 광고ID, IP 광고 제공
Google (Firebase) 기기정보, 앱 사용통계 서비스 분석

5. 이용자의 권리

🚫 광고 개인화 거부

기기 설정에서 광고 ID 재설정 또는 추적 거부가 가능합니다:

  • Android: 설정 → Google → 광고 → 광고 ID 삭제
  • iOS: 설정 → 개인정보 보호 → 추적 → 앱 추적 허용 끄기
